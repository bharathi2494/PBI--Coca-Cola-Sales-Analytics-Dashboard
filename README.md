# Coca-Cola Sales Dashboard Project (using Microsoft Power BI)
## Project Overview:
This project involves creating an interactive sales dashboard for Coca-Cola beverages using business intelligence tools (e.g., Power BI). The dashboard provides comprehensive insights into total sales, key influencers of profit, and sales distribution across the United States. It is designed to support data-driven decision-making for sales and marketing teams.
## Dataset used
- <a href= "https://github.com/bharathi2494/PBI--Coca-Cola-Sales-Analytics-Dashboard/blob/main/CocoCola-Dataset.xlsx">Dataset</a>
## Key Features:
- Total Sales Overview: Displays overall revenue and key sales metrics for different beverage brands including Coca-Cola, Diet Coke, Sprite, Powerade, Dasani Water, and Fanta.
- Brand-wise Performance: Breaks down sales, units sold, average price, operating profit, and operating margin for each beverage brand.
- Key Influencers Analysis: Identifies factors influencing operating profit. For example, increases in total sales or specific brands (e.g., Coca-Cola, Dasani) are shown to positively affect profit.
- Geographical Insights: Visualizes sales by state using an interactive map to help identify high-performing regions and areas for growth.
- Date Filtering: Allows filtering by invoice date range for dynamic analysis of sales trends over time.
